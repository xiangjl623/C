结合大学生 C++ 程序设计期末考试的要求，整理常规题型（选择题、填空题、问答题、纠错题、编程题等）。参考答案单独存放，按题型分文件输出到本目录（`languages/cpp/exams/`）；参考答案需包含考点解析和思路整理，方便新手复习。

**考点范围**：对齐本仓库 [C++ 教程](../guides/) 01～15 篇及 [语法速查](../references/C++常用语法汇总.md)。建议先修 [C 语言](../../c/)。

**题型与数量（首批）**：

- 选择题（50 道）：程序结构、引用与指针、函数重载、类与对象、继承多态、模板与 STL、智能指针等
- 填空题（50 道）：关键概念和语法细节
- 问答题（50 道）：C++ 核心原理与概念
- 纠错题（50 道）：常见语法错误、内存错误、OOP 错误
- 编程题（50 道）：iostream/string/vector、类设计、继承、STL、文件 I/O

**C++ 命题侧重**（相对 C 的增量）：

- 引用 vs 指针、nullptr、new/delete
- 函数重载、默认参数、inline
- class、构造/析构、virtual、override、多态
- std::string、std::vector、map、algorithm
- RAII、unique_ptr、shared_ptr

输出要求：
- 题目与答案分文件
- 答案含考点解析
- 编程题参考答案提供 C++17 风格可编译代码
