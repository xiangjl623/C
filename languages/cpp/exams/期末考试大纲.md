## 二、分篇考核知识点（摘要）

### 第 01 篇 C++ 入门

- C 与 C++ 关系；`.cpp` 源文件；`g++` 编译
- **能力**：写出最小 `iostream` 程序

### 第 02 篇 开发环境

- g++、`-std=c++17`、VS Code 任务；CMake 简介

### 第 03 篇 程序结构

- 头文件、`#pragma once`、namespace、`using`
- 多文件编译链接

### 第 04 篇 数据类型与引用

- `bool`、引用、`auto`、`constexpr`；`cin`/`cout`

### 第 05 篇 运算符

- new/delete、nullptr、static_cast；链接 C 运算符基础

### 第 06～07 篇 流程控制与循环

- if/switch 与 C 相同；`enum class`；**范围 for**

### 第 08 篇 数组与容器

- `std::string`、`std::vector`；与 C 数组对比

### 第 09 篇 函数

- **函数重载**、默认参数、inline、函数模板初探

### 第 10 篇 类与对象

- class、构造/析构、封装、const 成员、static 成员、this

### 第 11 篇 继承与多态

- 继承、virtual、override、纯虚函数、虚析构、对象切片

### 第 12 篇 模板与 STL

- 函数/类模板；vector、map、set；sort、find、count

### 第 13 篇 智能指针与 RAII

- unique_ptr、shared_ptr、weak_ptr；与 malloc/free 对比

### 第 14～15 篇 实战与路线

- 综合项目（类 + vector + 文件 I/O）；现代 C++ 学习路径
