# C++ 语言期末考试 — 编程题（50 道）

---

1. 用 cin/cout 输入两个整数，输出和、差、积、商。

2. 输入姓名（string），输出问候语。

3. 用 vector 存储 n 个整数，输出最大值。

4. 判断输入年份是否为闰年。

5. 用 string 反转字符串。

6. 统计 string 中元音字母个数。

7. 实现函数重载：print(int) 与 print(double)。

8. 实现 swap(int&, int&) 交换两整数。

9. 定义 Circle 类，含半径与 area() 方法。

10. 定义 Student 类，含 name、id 与 display()。

11. 用 vector<Student> 存储并遍历输出。

12. 派生 Dog 继承 Animal，重写 speak() 虚函数。

13. 用 map 统计单词出现次数（简化：以空格分隔）。

14. 用 sort 对 vector<int> 升序排序。

15. 用 find 在 vector 中查找指定值。

16. 文件写入三行文本到 out.txt。

17. 从 in.txt 读取并输出所有行。

18. unique_ptr 管理 int 动态对象并输出。

19. shared_ptr 共享对象，打印 use_count。

20. 模板函数 maximum(T,T) 返回较大值。

21. 类模板 Box<T> 存单个值并提供 get/set。

22. const 成员函数 getX() 返回坐标。

23. 静态成员计数创建对象个数。

24. 构造函数初始化列表初始化成员。

25. 析构函数输出对象销毁信息（RAII 演示）。

26. enum class 表示星期，switch 输出名称。

27. 范围 for 遍历 vector<string>。

28. auto 推导并输出变量类型大小（typeid/sizeof）。

29. 默认参数 greet(name, prefix="Hello")。

30. 深拷贝：含 char* 成员的类实现拷贝构造（简化用 string）。

31. 抽象类 Shape 派生 Rectangle 实现 area()。

32. 多态：基类指针数组调用 virtual draw()。

33. operator+ 重载复数类（实部虚部）。

34. friend 函数访问类 private 数据。

35. vector 去重（先 sort 再 unique erase）。

36. set 插入元素并遍历输出。

37. stringstream 解析 "123 456" 为两个 int。

38. getline 读整行含空格。

39. fstream 追加模式打开文件写一行。

40. 检查文件 is_open 再读写。

41. 类 BankAccount：deposit/withdraw/getBalance。

42. 继承 Person 派生 Employee 增加工号。

43. override 重写基类 virtual print()。

44. 纯虚函数接口 Printable 派生实现。

45. weak_ptr 观察 shared_ptr 是否过期。

46. make_unique 创建 vector<int> 并填充 1..n。

47. algorithm count 统计 vector 中某值个数。

48. 二维 vector 表示矩阵并输出。

49. StudentManager 简化版：add/list（单文件）。

50. 综合：读文件学生列表到 vector，按学号查找。
