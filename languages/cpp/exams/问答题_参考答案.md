# C++ 语言期末考试 — 问答题参考答案与解析

---

## 第 1 题

**题目：** 简述 C++ 与 C 的主要区别。

**参考答案：** C++ 在 C 基础上增加 OOP、模板、STL、异常、namespace 等；兼容大部分 C 语法。

**考点解析：** 见 C++ 教程对应章节。

---

## 第 2 题

**题目：** 什么是引用？与指针有何不同？

**参考答案：** 引用是别名，必须初始化，不能空，不能重绑；指针存地址，可为空，可改指向。

**考点解析：** 见 C++ 教程对应章节。

**题目：** 解释 RAII 原则及其意义。

**参考答案：** 构造获取资源，析构释放；保证异常安全，减少泄漏。

**考点解析：** 见 C++ 教程对应章节。

## 第 4 题

**题目：** 什么是函数重载？重载规则是什么？

**参考答案：** 同名不同参；参数类型或个数须不同，不能仅靠返回值区分。

**考点解析：** 见 C++ 教程对应章节。
