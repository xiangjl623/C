## 第 3 题

**题目：** 用 vector 存储 n 个整数，输出最大值。

**参考实现：**

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
int main() {
    int n; std::cin >> n; std::vector<int> v(n);
    for (int& x : v) std::cin >> x;
    std::cout << *std::max_element(v.begin(), v.end()) << std::endl;
    return 0;
}
```

**思路：** 见 [examples/chapter08+](../examples/) 及对应 guides。

## 第 6 题

**题目：** 统计 string 中元音字母个数。

```cpp
#include <iostream>
#include <string>
int main() {
    std::string s; std::cin >> s; int c=0;
    for (char ch : s)
        if (ch=='a'||ch=='e'||ch=='i'||ch=='o'||ch=='u'||
            ch=='A'||ch=='E'||ch=='I'||ch=='O'||ch=='U') c++;
    std::cout << c << std::endl;
    return 0;
}
```

## 第 9 题

**题目：** 定义 Circle 类，含半径与 area() 方法。

**参考实现：**

```cpp
#include <iostream>
class Circle {
    double r;
public:
    Circle(double radius): r(radius) {}
    double area() const { return 3.14159*r*r; }
};
int main(){ Circle c(5); std::cout<<c.area(); return 0; }
```
