## 编译运行

```bash
# 单文件编译（推荐 C++17）
g++ -std=c++17 -Wall -o out example.cpp

# Windows
out.exe

# Linux / macOS / WSL
./out
```

## 命名约定

- `example_NN.cpp`：章节内按序号排列的示例
