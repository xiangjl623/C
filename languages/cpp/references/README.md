## 使用建议

1. **先通读 01～04**：建立 C++ 程序结构与基本语法框架。
2. **掌握 02 中的引用与 iostream**：C++ 日常编程核心。
3. **08～09 面向对象**：类、继承、多态是 C++ 核心。
4. **10～12 STL**：vector、map、algorithm 日常使用频率高。
5. 与 C 速查对照学习：指针、数组等详见 C 语言速查。
