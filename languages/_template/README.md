## 使用说明

1. 复制 `_template/` 为 `languages/<lang>/`
2. 填写 `syllabus.md` 与本文 README
3. 在 `guides/` 各阶段目录下按 `01-topic-name.md` 命名新增教程
4. 在 [languages/README.md](../README.md) 注册新语言
