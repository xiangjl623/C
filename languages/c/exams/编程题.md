# C 语言期末考试 — 编程题（100 道）

---
1. 编写程序，输入两个整数，输出它们的和、差、积、商（整除）。

2. 输入圆的半径，计算并输出周长和面积（π取3.14159）。

3. 输入三个整数，输出最大值和最小值。

4. 判断输入的年份是否为闰年。

5. 输入成绩，输出对应等级（90+ A, 80-89 B, 70-79 C, 60-69 D, 否则 F）。

6. 输出100以内所有素数。

7. 输入n，计算1+2+...+n的和。

8. 输入n，计算n!（阶乘）。

9. 输出斐波那契数列前n项。

10. 输入一个整数，判断是否为素数。

11. 反转一个整数的各位数字并输出。

12. 输入整数n，输出其各位数字之和。

13. 打印九九乘法表。

14. 输入字符，判断是否为大写字母、小写字母或数字。

15. 统计输入字符串中字母、数字、空格的个数。

16. 将字符串中的小写字母转为大写。

17. 删除字符串中的空格。

18. 判断字符串是否为回文。

19. 实现字符串拷贝（不用strcpy）。

20. 实现字符串连接（不用strcat）。

21. 冒泡排序对n个整数升序排列。

22. 选择排序对数组排序。

23. 插入排序对数组排序。

24. 在有序数组中二分查找给定key。

25. 数组元素逆序存放。

26. 求数组中的最大值、最小值及下标。

27. 统计数组中各元素出现次数（元素范围小）。

28. 矩阵加法：两个m×n矩阵相加。

29. 矩阵乘法：A(m×k) × B(k×n)。

30. 转置矩阵。

31. 编写函数 swap 交换两个整数的值（用指针）。

32. 编写函数求两个整数的最大公约数（辗转相除）。

33. 编写函数求最小公倍数。

34. 编写递归函数求 n!。

35. 编写递归函数求斐波那契第n项。

36. 编写递归函数实现整数各位数字之和。

37. 用指针遍历数组并求和。

38. 用指针实现字符串长度计算。

39. 用指针实现字符串反转。

40. 动态分配数组，输入n个数并排序后输出。

41. 合并两个有序数组为一个有序数组。

42. 定义结构体 Student（学号、姓名、成绩），输入3名学生并输出。

43. 按成绩对学生结构体数组排序。

44. 统计文件中单词个数（假设单词以空格分隔）。

45. 复制文件内容到另一个文件。

46. 统计文件中某字符出现的次数。

47. 从文件读取整数直到EOF，求平均值。

48. 将结构体数组写入二进制文件再读回。

49. 实现单向链表的创建与遍历。

50. 在链表尾部插入结点。

51. 删除链表中值为x的结点。

52. 链表逆序。

53. 判断链表是否有环（思路题）。

54. 用栈判断括号是否匹配。

55. 用队列模拟约瑟夫环（n人报数出列）。

56. 实现简单计算器（+ - * / 两个数）。

57. 输入日期（年-月-日），输出是星期几（蔡勒公式）。

58. 将十进制整数转为二进制字符串输出。

59. 将二进制字符串转为十进制整数。

60. 求100以内完数（等于因子之和）。

61. 水仙花数：三位数等于各位立方和。

62. 打印菱形图案（输入层数）。

63. 打印杨辉三角前n行。

64. 猜数字游戏：随机数，用户猜，提示大/小。

65. 模拟掷骰子统计1000次各点数频率。

66. 求一元二次方程 ax²+bx+c=0 的实根。

67. 用牛顿迭代法求平方根。

68. 计算 e 的近似值（级数）。

69. 计算 sin(x) 的泰勒展开近似。

70. 统计一行文本中每个单词的长度。

71. 找出字符串中最长单词。

72. 字符串中替换所有某字符为另一字符。

73. 统计每个字符出现次数。

74. 去除字符串首尾空格。

75. 实现 atoi（字符串转整数）。

76. 实现 itoa（整数转字符串）。

77. 读入矩阵，求对角线元素之和。

78. 判断矩阵是否为对称矩阵。

79. 螺旋打印矩阵（进阶）。

80. 结构体日期比较哪个更早。

81. 职工信息：按工资排序并输出。

82. 图书信息管理系统：增删查（数组版）。

83. 学生成绩文件：读入、算平均分、写回。

84. 命令行参数求和：argc argv 整数相加。

85. 实现 mystrlen, mystrcpy, mystrcmp。

86. 用函数指针数组实现四则运算。

87. 统计二进制文件中0和1的位数（按字节）。

88. 加密：凯撒密码偏移k位。

89. 解密凯撒密码。

90. 简单通讯录：姓名电话，菜单驱动。

91. 求两个字符串最长公共子串（基础版）。

92. 数组轮转左移k位。

93. 找出数组中第二大的数。

94. 找出只出现一次的数（其他出现两次）。

95. 合并区间（简化版）。

96. 求数组连续子数组最大和（Kadane）。

97. 实现栈的数组版本（push/pop）。

98. 实现队列的数组版本。

99. 链表合并（两个有序链表）。

100. 删除链表重复元素。
