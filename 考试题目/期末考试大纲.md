- 动态内存：`malloc`、`calloc`、`realloc`、`free`

- 用指针实现动态数组

#### 教材常见错误浓缩（易错点清单）

| 类别 | 典型错误 |
|------|----------|
| 运算 | 整数除法 `5/2` 得 2 而非 2.5；`++i` 与 `i++` 混用 |
| 输入输出 | `scanf` 漏写 `&`；`%f`/`%lf` 与 float/double 不匹配 |
| 数组 | 下标越界；将数组大小误传给 sizeof 退化的指针参数 |
| 字符串 | `strcpy` 目标缓冲区不足；未留 `'\0'` 空间 |
| 指针 | 未初始化就解引用；释放后继续使用（悬空指针） |
| 内存 | `malloc` 后未 `free`；重复 `free` |
| 文件 | 未检查 `fopen` 返回值；忘记 `fclose` |
| 结构体 | 混淆 `.` 与 `->`；链表删除时指针链接错误 |
